## 🎮 Usage Examples

### Build a Coding Assistant
```typescript
import { CodingAgent } from '@/lib/ai-agents';

const agent = new CodingAgent({
  model: 'claude-3.5-sonnet',
  adapter: 'coding-v2',
  deployment: 'aws-lambda'
});

await agent.train();
await agent.deploy();
```

### Extract SaaS Intelligence
```typescript
import { SaaSExtractor } from '@/lib/saas-database-extractor';

const extractor = new SaaSExtractor();
const pricing = await extractor.scrapePricing('competitor.com');
console.log(pricing); // Structured pricing data
```

### Deploy GOAT Workflow
```bash
cd goat-deployment
./goat-launcher.py --workflow=dev --cloud=modal
```
